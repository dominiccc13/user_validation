# Project Title (Update)

A simple web app that allows for user registration, verification, and login. Utilizes a C# backend, SQLite database, and password hashing. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Visual Studio
2. Clone the repository
3. Load the project into Visual Studio

Instructions for using the software:

1. Run with debugging (ctrl + F5)
2. This will open a window that is serving all of the files via localhost
3. Feel free to register as a user, confirm your email, and then login

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions: 
*All are included in the github repo, but you will need these nuget packages:*

* DotNetEnv by DotNetEnv
* Microsoft.Data.Sqlite by Microsoft
* SendGrid by Elmer Thomas, Twilio DX Team
